O que é o Python?

Python é uma linguagem de programação de alto nível, interpretada, de código aberto e multiplataforma. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991. O nome "Python" foi inspirado pelo programa de televisão britânico "Monty Python's Flying Circus," do qual Guido era fã.

Por que usar Python?

Python em essência tem uma sintaxe muito simples. A mesma lógica que necessita 7 linhas em linguagem C++, requer apenas 3 no python. Ter um código menor significa que precisa de menos espaço, menos tempo, e é bem recebido entre os programadores, porque requer menos tempo para retrabalhar e corrigir. A linguagem cumpre todos os parâmetros de legibilidade. Para se auto-ajudar a linguagem tem vários built-ins e bibliotecas que o tornam fáceis de compreender.

Python é uma linguagem usada no mercado?

Incontáveis empresas conceituadas utilizam Python em diversas áreas de seu sistema. Algumas delas são: Youtube, Instagram, Google, NASA, Spotify e Netflix. O seu uso vai desde a parte web, com a framework Django, até estruturas de backend com banco de dados, infraestrutura e automação.

Frameworks para estudar no Python:

Flask

Framework web mais leve e flexível em comparação com o Django. Ele fornece apenas o básico para a construção de aplicativos web.

Django

Framework de desenvolvimento web de alto nível que incentiva o desenvolvimento rápido e limpo. Ele fornece muitas ferramentas e recursos para ajudar os desenvolvedores

FastAPI

Framework moderno para construção de APIs web de alta performance, combina a facilidade de uso e a sintaxe concisa do Python com a velocidade comparável a linguagens de programação mais rápidas, como o Go.
